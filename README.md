# Web Socket Project

A real-time chat application built with **Node.js**, **Express**, **Socket.IO**, and **React**. This project demonstrates how to implement WebSocket communication for live messaging between multiple clients in designated chat rooms.

---

## Technologies
- **Server-side:** Node.js, Express, Socket.IO  
- **Client-side:** React, JavaScript, HTML, CSS  
- **Deployment:** Vercel (frontend), Render (server)

---

## Features
- Real-time messaging between users  
- Multiple chat rooms support  
- Event-driven communication using WebSockets  
- Cross-origin requests handled via CORS  
- Lightweight and easy to deploy
